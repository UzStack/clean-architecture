### 1-2 Hafta: Clean Architecture
**Maqsad:** Toza arxitektura asoslarini o'zlashtirish

- [ ] Qatlamlar (Entities, Use Cases, Controllers) bilan tanishish

---

### 3 Hafta: SOLID Prinsiplari
**Maqsad:** OOP asoslarini chuqur o'rganish

- [ ] Har bir SOLID prinsipini o'rganish
- [ ] Kod misollarini tahlil qilish
- [ ] **Amaliyot:** Mavjud loyihani refactor qiling
  - God Object'larni topish va bo'lish
  - Dependency Injection qo'llash
  - Strategy Pattern qo'llash

**Natija:** Yaxshi OOP kod yoza olasiz

---

### 4-5 Hafta: Design Patterns
**Maqsad:** Umumiy muammolarni yechishni o'rganish

**Hafta 4: Creational + Structural**
- [ ] Singleton, Factory, Builder
- [ ] Adapter, Decorator, Facade
- [ ] **Amaliyot:** 
  - Payment system (Factory + Strategy)
  - Logging system (Singleton + Decorator)

**Hafta 5: Behavioral**
- [ ] Observer, Command, Strategy
- [ ] Template Method, Chain of Responsibility
- [ ] **Amaliyot:**
  - Notification system (Observer)
  - Order processing (Command + Chain)

**Natija:** Pattern'larni real loyihalarda qo'llaysiz

---

### 6-7 Hafta: Domain-Driven Design
**Maqsad:** Murakkab biznes logikani modellash

**Hafta 6: Tactical DDD**
- [ ] Entities va Value Objects
- [ ] Aggregates
- [ ] Domain Services
- [ ] **Amaliyot:** Banking system
  - Account (Aggregate Root)
  - Money (Value Object)
  - Transfer (Domain Service)

**Hafta 7: Strategic DDD**
- [ ] Bounded Contexts
- [ ] Ubiquitous Language
- [ ] Context Mapping
- [ ] **Amaliyot:** Multi-context e-commerce
  - Sales Context
  - Inventory Context
  - Shipping Context

**Natija:** Complex domain'larni modellaysiz

---

### 8 Hafta: Testing
**Maqsad:** Professional test yozish

- [ ] Unit Testing
- [ ] Integration Testing
- [ ] TDD practice
- [ ] Mocking frameworks
- [ ] **Amaliyot:**
  - Barcha proyektlaringizga test yozing
  - Test coverage 80%+ bo'lishi kerak
  - TDD bilan yangi feature yozing

**Natija:** Kod sifati yuqori bo'ladi

---

### 9 Hafta: System Design

- [ ] **Amaliyot:**

  - Instagram-like app design

**Natija:** System design interview'lardan o'tasiz
